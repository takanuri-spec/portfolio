## 🛠️ 技術スタック

本ポートフォリオサイトは、フレームワークに依存せず、軽量かつ高速に動作するようピュアなWeb標準技術で構築されています。

- **HTML5**: セマンティックなマークアップ
- **CSS3** (Vanilla CSS): 
  - CSS Variables（カスタムプロパティ）によるテーマ管理
  - CSS Grid / Flexbox によるレスポンシブなレイアウト設計
  - `backdrop-filter` やホバーエフェクトによるグラスモーフィズム・マイクロインタラクション
- **JavaScript** (Vanilla JS): 
  - DOM操作と要素の段階的なフェードイン（Fade-in）アニメーション制御
- **Google Fonts**: 「Outfit」フォントセットを使用

## 📁 主要ファイル構成

- `index.html`: メインのレイアウトおよび各プロジェクトへのリンク構造
- `style.css`: サイト全体のスタイリング、アニメーション定義、レスポンシブ対応
- `script.js`: 要素表示時のアニメーション遅延等のスクリプト
- `assets/icons/`: 各サービスのサムネイル画像
